## 4. 小游戏详细介绍（120-300字）

> 请用一段文字详细介绍游戏作品内容，包括但不限于：游戏背景、场景、主要情节、玩法、功能（系统）、主要特点、游戏使用方法等。

### 推荐方案 A（推荐）

```
《国风轻玩》是一款以中国传统文化为主题的益智小游戏合集，水墨国风画面，带你沉浸式感受汉字与诗词的魅力。

游戏包含三大核心玩法：
【找成语】在汉字矩阵中寻找隐藏的成语，从简单的4×4方阵到困难的5×7方阵，考验你的成语储备和眼力。
【汉字找不同】在大量相同汉字中找出唯一不同的那一个，越往后难度越高，锻炼你的专注力和观察力。
【诗词连线】将诗词的上句与下句正确配对连线，重温经典古诗词，体验对句成诗的乐趣。

每关设有计时挑战和星级评分，支持简单、一般、困难三档难度，适合各年龄段玩家。国风水墨视觉设计，让你在轻松益智中领略传统文化之美。
```
（252字）

### 备选方案 B

```
《国风轻玩》是一款融合中国传统文化元素的益智小游戏合集，以水墨国风为视觉风格，提供找成语、汉字找不同、诗词连线三大经典玩法。
```
