## 🎮 游戏概览

### 三款核心游戏（v1.4）

| 游戏名称 | 计分模式 | 计分公式 | 难度 |
|---------|---------|---------|------|
| 📜 找成语 | 速度+准确 | `100 + 剩余×2 - 已用×0.5 - 错误×3` | ⭐⭐⭐ |
| 🔍 汉字找不同 | 准确+速度 | `100 - 已用×0.8 - 错误×15` | ⭐⭐ |
| 🖌️ 诗词连线 | 准确+速度 | `100 - 已用×0.6 - 错误×10` | ⭐⭐⭐ |

### 核心特性

- ✅ **首次进入**：显示计分规则和开始按钮
- ✅ **连续玩关**：点击"下一关"后自动开始
- ✅ **智能计分**：时间奖励 + 耗时惩罚 + 错误惩罚
- ✅ **保底机制**：最低保证 10 分

### 新人上手指南

1. **了解项目** → 阅读 [产品需求文档](./产品需求文档.md)
2. **学习规则** → 阅读 [积分规则文档](./积分规则文档.md)
3. **查看更新** → 阅读 [功能更新日志](./功能更新日志.md)
4. **开始开发** → 阅读 [前端开发文档](./前端开发文档.md) 或 [后端开发文档](./后端开发文档.md)
5. **功能测试** → 按照 [测试指南](./测试指南.md) 执行

### 开发环境搭建

#### 前端（小程序）
```bash
1. 安装微信开发者工具
2. 打开 minigame 目录
3. 配置 AppID
4. 点击编译
```

#### 后端（Node.js）
```bash
cd server
npm install
npm run dev
```

## 📦 项目结构

```
vx-game/
├── minigame/              # 小程序前端
│   ├── pages/
│   │   ├── index/         # 首页
│   │   ├── game/          # 游戏页（含开始按钮）
│   │   └── result/        # 结果页
│   ├── gameLogic/
│   │   ├── wordFind.js    # 找成语逻辑
│   │   ├── charDiff.js    # 汉字找不同逻辑
│   │   └── poetryConnect.js # 诗词连线逻辑
│   └── utils/             # 工具函数
├── server/                # Node.js 后端
└── docs/                  # 📚 文档（当前目录）
    ├── README.md          # 👈 你在这里
    ├── 积分规则文档.md      # 权威规则文档
    ├── 功能更新日志.md      # 版本更新历史
    ├── 测试指南.md          # 测试用例
    ├── 计分规则快速参考.md  # 快速查询
    ├── 前端开发文档.md
    ├── 后端开发文档.md
    └── 产品需求文档.md
```
