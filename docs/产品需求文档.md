## 📱 产品概述

### 产品名称
国风轻玩合集

### 产品定位
一款以中国传统文化为主题的微信小程序益智游戏合集，包含找成语、汉字找不同、诗词连线三款核心玩法。

### 目标用户
- **主要用户**：18-45 岁，对传统文化感兴趣的用户
- **次要用户**：学生群体（小学至高中）
- **潜在用户**：中老年用户（简单难度）

### 产品价值
- **用户价值**：轻松娱乐 + 传统文化学习
- **商业价值**：广告变现 + 潜在会员增值服务
- **社会价值**：传播中国传统文化

## 🎯 核心功能

### 1. 首页 - 游戏选择

#### 功能描述
展示三款游戏的入口卡片，用户点击进入对应游戏。

#### 界面元素
- **标题区域**：
  - 主标题：国风轻玩合集
  - 副标题：— 经典国风益智小游戏 —
  
- **游戏卡片**（2列3行布局）：
  - 卡片1：📜 找成语 ⭐⭐⭐
  - 卡片2：🔍 汉字找不同 ⭐⭐
  - 卡片3：🖌️ 诗词连线 ⭐⭐⭐
  
- **卡片状态**：
  - 已解锁：正常显示 + 右上角绿点
  - 未解锁：半透明遮罩 + 🔒图标
  
- **底部提示**：点击卡片开始游戏

#### 交互逻辑
1. 首次进入，只有"找成语"解锁
2. 点击已解锁卡片 → 进入游戏（第1关）
3. 点击未解锁卡片 → 提示"该游戏尚未解锁"

#### 解锁规则
- 找成语：默认解锁
- 汉字找不同：找成语通过3关后解锁
- 诗词连线：汉字找不同通过3关后解锁

### 2. 游戏页面

#### 2.1 通用元素

**顶部标题栏：**
- 左侧：返回按钮
- 中间：游戏图标 + 游戏名称
- 右侧：关卡标签（第X关）

**倒计时进度条：**
- 位置：标题栏下方
- 样式：
  - 剩余时间 > 50%：绿色
  - 剩余时间 30-50%：橙色
  - 剩余时间 < 30%：红色
- 显示：剩余秒数（右侧）

**游戏区域：**
- 居中显示
- 根据屏幕自适应缩放

**底部按钮：**
- 💡 提示按钮（居中）

---

#### 2.2 找成语

**游戏机制：**
1. 屏幕显示若干汉字（正确成语4字 + 干扰字）
2. 玩家依次点击4个字组成成语
3. 选中的字显示红色背景 + 序号标记
4. 选满4个字自动判断：
   - 正确：立即跳转结算页
   - 错误：提示"答案不对，再试试"，1.5秒后自动清空

**难度配置：**
- 简单（1-3关）：4x4=16字，20秒
- 一般（4-6关）：5x5=25字，30秒
- 困难（7-10关）：5x7=35字，40秒

**视觉设计：**
- 字体：宋体/楷体，带笔触质感
- 背景：米色宣纸质感
- 选中效果：朱砂红渐变 + 白色文字
- 已选预览：顶部显示"已选：XXX"

---

#### 2.3 汉字找不同

**游戏机制：**
1. 屏幕显示大量相同的字，其中1个不同
2. 玩家点击不同的字
3. 点击正确：立即跳转结算页
4. 点击错误：字抖动，扣分

**难度配置：**
- 简单（1-3关）：3x3=9字，差异明显，20秒
- 一般（4-6关）：4x4=16字，笔画差异小，25秒
- 困难（7-10关）：5x5=25字，极度相似，30秒

**视觉设计：**
- 字体：统一大字号，黑色
- 背景：浅色简洁
- 错误反馈：红色闪烁

---

#### 2.4 诗词连线

**游戏机制：**
1. 左右两列诗句，玩家连接上下句
2. 点击左边诗句 → 点击右边诗句 → 形成连线
3. 连对：绿色实线
4. 连错：红色虚线，1秒后消失
5. 全部连对：立即跳转结算页

**难度配置：**
- 简单（1-3关）：3对，30秒
- 一般（4-6关）：5对，45秒
- 困难（7-10关）：8对，70秒

**视觉设计：**
- 字体：楷体，带古风韵味
- 连线：贝塞尔曲线，优雅流畅
- 背景：水墨画风格

---

### 3. 结算页面

#### 功能描述
展示本局游戏结果，引导用户下一步操作。

#### 界面元素

**成功状态：**
- 顶部：✅ 通关成功
- 中间：
  - 得分：XX 分（大字号，金色）
  - 用时：XX 秒
  - 星级：⭐⭐⭐（根据分数）
- 底部按钮：
  - 【下一关】（主按钮）
  - 【返回首页】（次按钮）

**失败状态（时间到）：**
- 顶部：⏰ 时间到
- 中间：
  - 提示文字：再试一次吧
  - 鼓励语：距离成功就差一点点
- 底部按钮：
  - 【重新挑战】（主按钮）
  - 【返回首页】（次按钮）

#### 星级评定
- ⭐ 1星：60-79分
- ⭐⭐ 2星：80-89分
- ⭐⭐⭐ 3星：90-100分

## 🔄 用户流程

### 新用户首次使用
```
打开小程序 
  ↓
首页（只有找成语解锁）
  ↓
点击"找成语" 
  ↓
第1关（简单）
  ↓
完成 → 结算页 → 下一关
  ↓
完成第3关 → 解锁"汉字找不同"
  ↓
返回首页 → 看到新游戏解锁提示
```

### 老用户回流
```
打开小程序
  ↓
首页（显示三个游戏进度）
  ↓
选择任意游戏继续
```
