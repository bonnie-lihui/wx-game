## 🎮 三个核心游戏

### 1. 找成语 (wordFind)

#### 游戏机制
- 屏幕显示若干汉字（正确成语 + 干扰字）
- 玩家依次点击 4 个字组成成语
- 选对 4 个字后自动跳转结算页

#### 难度配置
```javascript
// gameLogic/wordFind.js
var DIFFICULTY_CONFIG = {
  easy:   { cols: 4, rows: 4, cellSize: 72 },   // 16 字，简单
  normal: { cols: 5, rows: 5, cellSize: 64 },   // 25 字，一般
  hard:   { cols: 5, rows: 7, cellSize: 52 },   // 35 字，困难
};
```

#### 时间限制
- 简单：20 秒
- 一般：30 秒
- 困难：40 秒

#### 关键方法
```javascript
// 初始化矩阵
WordFindLogic.prototype.initMatrix = function()

// 点击格子
WordFindLogic.prototype.tapCell = function(row, col)
// 返回: { success, isComplete, isFull, isCorrect }

// 绘制游戏界面
WordFindLogic.prototype.draw = function(ctx, offsetX, offsetY)

// 判断点击位置
WordFindLogic.prototype.getCellByPoint = function(x, y)
```

### 2. 汉字找不同 (charDiff)

#### 游戏机制
- 大量相同汉字中有 1 个不同
- 玩家点击不同的字
- 找到后立即跳转结算页

#### 难度配置
```javascript

  easy:   { cols: 3, rows: 3, cellSize: 100 },  // 9 字，差异明显
  normal: { cols: 4, rows: 4, cellSize: 80 },   // 16 字，笔画差异小
  hard:   { cols: 5, rows: 5, cellSize: 64 },   // 25 字，极度相似
};
```

#### 时间限制
- 简单：20 秒
- 一般：25 秒
- 困难：30 秒

#### 关键方法
```javascript
// 初始化矩阵
```

### 3. 诗词连线 (poetryConnect)

#### 游戏机制
- 左右两列诗句，匹配上下句
- 点击左边 → 点击右边形成连线
- 全部连对后立即跳转结算页

#### 难度配置
```javascript
// gameLogic/poetryConnect.js
// 简单：3 对
// 一般：5 对
// 困难：8 对
```

#### 时间限制

- 困难：70 秒

#### 关键方法

#### Canvas 初始化
```javascript
initCanvas() {
  const query = wx.createSelectorQuery().in(this);
  query.select('#gameCanvas')
    .fields({ node: true, size: true })
    .exec((res) => {
      const canvas = res[0].node;
      const ctx = canvas.getContext('2d');
      const dpr = wx.getSystemInfoSync().pixelRatio || 2;
      canvas.width = width * dpr;
      canvas.height = height * dpr;
      ctx.scale(dpr, dpr);
      this.ctx = ctx;
      this.draw();
    });
}
```

## 🎯 关键交互逻辑

### 自动跳转规则

所有游戏完成后**立即跳转**结算页（0 延迟）：

```javascript
// 找成语：选满 4 个字后
if (ret && ret.isFull) {
  if (ret.isCorrect) {
    self.goResult(true);  // 正确立即跳转
  } else {
    // 错误提示 + 自动清空
    wx.showToast({ title: '答案不对，再试试', icon: 'none' });
    setTimeout(() => {
      self.logic.selected = [];
      self.draw();
    }, 1500);
  }
}

// 汉字找不同：找到答案
if (ret.hit) {
  self.goResult(true);
}

// 诗词连线：全部连对
if (res.isComplete) {
  self.goResult(true);
}
```

## 🐛 常见问题解决

### 1. Canvas 模糊问题
```javascript
// 解决方案：设置正确的 DPR
const dpr = wx.getSystemInfoSync().pixelRatio || 2;
canvas.width = width * dpr;
canvas.height = height * dpr;
ctx.scale(dpr, dpr);

// CSS 字体抗锯齿
-webkit-font-smoothing: antialiased;
-moz-osx-font-smoothing: grayscale;
```

### 2. 触摸事件偏移
```javascript
// 确保 draw() 和 handleTap() 中的坐标计算一致
// areaTop 必须相同
const areaTop = titleBarH + 28;

// nativeW 和 nativeH 必须动态计算
if (this.logic && this.gameId === 'wordFind') {
  nativeW = this.logic.cols * (this.logic.cellSize + 6) - 6;
  nativeH = this.logic.rows * (this.logic.cellSize + 6) - 6;
}
```

### 3. 坐标转换
```javascript
// 屏幕坐标 → 游戏区域本地坐标
const scale = Math.min(availW / nativeW, maxH / totalNativeH, 1.15);
const offsetX = (w - nativeW * scale) / 2;
const offsetY = areaTop + extraTop * scale;
const localX = (x - offsetX) / scale;
const localY = (y - offsetY) / scale;
```

## 🚀 性能优化建议

### 1. Canvas 绘制优化
- 避免全屏重绘，只更新变化区域
- 倒计时进度条单独重绘（不包含秒数文字）

### 2. 图片资源优化
- 使用 WebP 格式
- 压缩图片大小
- 懒加载非首屏图片

### 3. 代码优化
- 减少不必要的 setData
- 避免频繁的 DOM 操作
- 使用节流/防抖处理高频事件
