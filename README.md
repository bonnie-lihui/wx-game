# 国风轻玩合集

一款以中国传统文化为主题的微信小程序益智游戏合集。

---

## 🎮 游戏介绍

### 三款核心游戏
- 📜 **找成语**：从混乱的汉字中找出正确的成语
- 🔍 **汉字找不同**：在大量相同的字中找到唯一不同的字
- 🖌️ **诗词连线**：匹配正确的诗词上下句

## 📁 项目结构

```
vx-game/
├── minigame/          # 小程序前端
│   ├── pages/        # 页面
│   ├── gameLogic/    # 游戏逻辑
│   └── utils/        # 工具函数
├── server/           # Node.js 后端
│   ├── routes/       # 路由
│   └── db/           # 数据库

    ├── 后端开发文档.md
    ├── 产品需求文档.md
    └── 积分规则文档.md
```

## 🚀 快速开始

### 前端（小程序）
```bash
# 1. 打开微信开发者工具
# 2. 导入 minigame 目录
# 3. 配置 AppID
# 4. 点击编译
```

### 后端（Node.js）
```bash
cd server
npm install
npm run dev
```

## 🎯 核心特性

- ✅ **满分统一**：所有游戏满分 100 分
- ✅ **自动跳转**：完成后立即跳转结算页
- ✅ **国风设计**：水墨画风 + 朱砂红配色
- ✅ **难度分级**：简单、一般、困难三档
- ✅ **容错机制**：保底分 10 分
